# RoomBeat Roadmap

## Phase 1: Core MVP, 4-6 weeks

- Auth and user profiles.
- YouTube search and video metadata proxy.
- Room creation and joining.
- Socket.io room gateway.
- Redis-backed queue.
- Host playback controls.
- Joiner latency compensation.
- Basic chat and sound reactions.
- Kinetic dark room shell with adaptive thumbnail theming hooks.
- Social queue metadata, requester avatars, and democratic vote contracts.
- Server-side YouTube search, metadata lookup, and playlist import endpoints.
- Bottom navigation, mini-player, search, library, rooms, and artist page shells.

## Phase 2: UX and SEO, 2-3 weeks

- Next.js SSR landing and discovery surfaces.
- Dynamic room metadata and OpenGraph images.
- Discovery tab and room previews.
- Mobile room UI polish.
- Share links and join-by-code improvements.
- Featured platform channels, Beat Rate trending, friend activity, and mood filters.
- Synchronized lyrics pane and host-approved jump suggestions.
- Host control center for queue rights, DJ style, co-hosts, and moderation.

## Phase 2.5: Social Experience Engine, 3-4 weeks

- Audio shoutouts and real-time DJ talk-over mode.
- Song battles and live music quizzes.
- After-party highlights and auto-generated shared playlists.
- Music Taste DNA visualization, DJ levels, badges, and unlockable room themes.
- Smart home lighting integration proof-of-concept.

## Phase 3: AI Contextual Recommendations, 2-3 weeks

- Weather and location API integration.
- Contextual YouTube playlist suggestions.
- Recommendation worker abstraction.
- Pinecone-ready vector indexing interface.
- Environmental AI based on weather, local time, and physical activity signals.

## Phase 4: Monetization, 3-5 weeks

- Coins and gems ledger.
- Digital gifts.
- Host support events.
- Fraud/rate limits.
- Reporting and admin views.
